# 混乱 DLC / Chaos DLC — 玩法规格(v1,owner 拍板版)

> **状态**:规则已与 owner 逐条锁定,待 polish + 借鉴 party game 继续拓展。仅出规格、不含代码。
> **与前文关系**:本方案是 owner 最终拍板的方向,**取代** `FINAL.md` 里"经典 / 对抗 / 乱斗"三档的建议——最终**只做两个模式:经典 + 混乱 DLC**。`FINAL.md` 里那个"打得好就让领先者多摸一张"的基础塞牌点子,**并入**本模式的「砸牌 +N」能力卡。
> **标注**:`【owner】`= owner 拍板的规则;`【补】`= 我补的细节/缺省值,可推翻。

---

## 0. 模式结构
- **经典**(默认):零干扰,放松拼脑子。
- **混乱 DLC**(开房 opt-in):本文档。
- `【owner】` 胜负不变:**先出完手牌者赢**;**无积分**;**能力卡不算手牌、不公开张数**。

## 1. 牌的三态模型(核心不变量)
`【owner】` 一张**普通牌**永远只在三个地方之一:**手里 / 桌上 / 牌库里**。
- **没有弃牌堆、没有"销毁"**(弃牌堆只给能力卡用,见 §3)。
- 任何被捣乱挪掉的普通牌**立刻回牌库重洗**。
- 推论:106 张牌是个**封闭循环**,游戏不会因"牌没了"而卡死;**赢的唯一方式仍是清空手牌**。

据此,所有"拿走牌"的效果都表述为**回牌库**;所有"给牌"的效果都表述为**从牌库摸**。

## 2. 公共随机转盘(混乱引擎)
`【owner】` **全程只有一个转盘**(不拆福/祸盘、不做秒级小游戏——线上秒级互动不做)。所有触发都转这同一个盘。

**触发条件(`【补】`每回合最多触发 1 次):**
1. `【owner】` 有人打出**「转盘卡」**(一种能力卡);
2. `【owner】` 某人**一回合往桌上贡献的牌 > 7 张**(打一手特别大,就"惊动转盘");
3. `【owner C】` 某人一回合**做成一个四色大组**(高光操作自带彩头/风险);
4. `【owner C】` 某人手牌**首次降到 ≤ 3 张**(收尾突刺,制造混乱终盘)。

> `【补】` 注:joker(小丑)**不挂转盘**——它归 §6「小丑炸弹」单独管,避免一张牌被双重收费。触发越多=转盘转得越勤=越混乱,这是个旋钮;`触发③④`想关就关。

**转一下 = 随机对象 × 随机行动**(对象与行动配对,避免语义错乱):

| 随机对象 | 随机行动 | 祸/福 |
|---|---|---|
| 某玩家(随机抽中) / 所有玩家 | 从牌库摸 1–3 张(张数随机) | 祸(手牌变多,离赢更远) |
| 某玩家 / 所有玩家 | 随机把你一张手牌**丢回牌库** | 福(手牌变少,离赢更近) |
| 某玩家 / 所有玩家 | 手牌全丢回牌库,**重摸相同数量** | 看运气(换一手) |
| 公共牌桌 | 从牌库随机摆**一套牌(长度随机 3–6)**上桌 | 机会(桌上凭空多料,人人可用) |
| 公共牌桌 | 从桌上随机**收走几套牌**回牌库 | 打击(可能拆掉别人想用的料 / 卷走 joker) |

> **设计注记(涌现张力)**:`触发②`让"打一手 8 张以上的大牌"既爽(清一堆手牌)又有风险(可能转出一个砸向所有人甚至自己的事件)——"要不要冒险打这超大一手"成了混乱里少有的一个真决定。

## 3. 能力卡
**怎么拿:**
- `【owner】` **纯随机发,不看技术。** 摸普通牌就是摸普通牌,与发能力卡互不相干。
- `【owner】` 缺省:开局每人发 2 张;之后**每回合开始 30% 概率发 1 张**(能力卡不能太多)。
- `【owner】` **没有手上限;别人不知道你有几张(完全隐藏)。**
- `【owner】` **全局铁律:所有"因惩罚而摸的牌"一律只摸普通牌,绝不摸能力卡**(砸牌 +N、逼牌、诈唬、小丑炸弹、转盘抽牌……全适用)。能力卡只来自上面的随机发放。

**怎么出:**
- `【owner】` 你的回合**想出几张就出几张,不限张数**(也可一张不出)。

**寿命:**
- `【owner】` 能力卡**一次性**:用完进**能力卡弃牌堆,且永不洗回,用完就没了**。
- **涌现的好处(白送的收尾节奏)**:能力卡是**全局有限资源**,边发边耗、会自然枯竭 → **越到后期卡越少 → 混乱自动收敛、收尾回归接近经典**,不会吵到最后一张牌。

### 能力卡清单(含稀有度,`【owner】` 白 / 蓝 / 金)
缺省整副 **28 张**;稀有度 = 牌库里的数量(白多金少),随机抽自然就白多金少。抽到**金卡**=一个"高光时刻"。

| 卡 | 稀有度 | 张数 | 大白话效果 |
|---|---|---|---|
| **砸牌 +2** | 白 | 5 | 指定一人,强制他从牌库摸 2 张。可 UNO 式转移。 |
| **护盾** | 白 | 5 | 留身上,被任意卡指定时自动抵消一次(含整条砸牌链)。 |
| **窥探** | 白 | 3 | Peek:看某人手牌,**本轮持续可见(可折叠面板)**,不是只看几秒。 |
| **砸牌 +3** | 蓝 | 3 | 指定一人摸 3 张。可 UNO 式转移。 |
| **逼牌** | 蓝 | 2 | 指定一人,下回合必须出≥1张,否则摸 3 张。 |
| **转盘卡** | 蓝 | 2 | 打出立刻转一次公共转盘(=触发①)。 |
| **大风吹** | 蓝 | 2 | 所有人同时把手里一张牌传给左边(对称,不针对单人)。 |
| **砸牌 +4** | 金 | 2 | 指定一人摸 4 张。可 UNO 式转移。 |
| **跳过** | 金 | 2 | 指定一人,下回合跳过。 |
| **封族** | 金 | 2 | 锁住桌上某一已成组的牌,`【补】`缺省 2 回合内谁都不能拆/动。 |

> 白 13 / 蓝 9 / 金 6 = 28。最强的(+4 / 跳过 / 封族)做金卡稀有;进攻牌(砸牌共 10)用**护盾 5**对冲——这是"砸牌链不封顶"的泄压阀。

### 砸牌的 UNO 式转移(`【owner】` 方向,`【补】` 细则)
被砸时,你手里若也有**砸牌**,可立刻打出,把**累计张数转给你指定的下一个人并叠加**(例:被 +2,你出 +3 → 下家吃 +5),一路传到**有人没砸牌/不接,由他一次性全摸下**。
- **护盾**能把整条砸牌链对你这一下整个抵掉。
- `【owner】` **不设链上限**(理由:手牌越多在 Rummikub 里反而越不容易卡手,被砸不是死亡螺旋,只是拖长进度)。

### 能力卡诈唬(Coup 式,`【owner B】`)
卡不公开张数,正好接得上诈唬。**任何卡都能盖着出。**
- 出能力卡时,可**盖着出**并**报一个名**(例:"这张是 砸牌 +4,砸老王")。
- **谁能质疑**:`【owner】` **指定单人的卡 → 只有目标能质疑;非指定卡 → 全场可质疑**(`【补】` 多人想质疑时先到先得)。质疑用现有软时钟,超时**默认放过**(非手速)。
- 结算(注意:本作**手牌越少越接近赢**,所以"消灭手牌"是奖励):

  | 情况 | 质疑者 | 诈唬人 |
  |---|---|---|
  | **质疑成功**(他确实在骗) | **随机消灭自己 1 张手牌**回牌库(奖励) | **摸 2 张**(罚)+ 卡作废亮出 |
  | **质疑失败**(他其实诚实) | **摸 2 张**(罚) | 卡照报生效 |

- **没人质疑** → 照报生效、**不亮牌**(诈唬蒙混成功)。
- `【owner】` 上述摸牌**只摸普通牌,不摸能力卡**(见全局铁律)。
- **设计注记**:用"手牌数"这个胜负货币当赌注——**质疑对了直接朝胜利前进一步(掉 1 张)**,错了 +2 张倒退;自我平衡,质疑成了真心理博弈。
- 深度:把一张没用的卡盖着出、谎称 砸牌 +4 吓人;也可真出 +4 赌对方来质疑一个真话。

## 4. 小丑炸弹(烫手山芋 / Joker Bomb)
`【owner D】` 把**小丑(joker)所在的组变成"活炸弹"**——它最强,但碰它就是赌命。

- **埋雷**:谁**把小丑打上桌**,就在桌上种下一个"含 joker 的活组"。(`【补】` 种下那一下不掷;之后才危险。)
- **引信(`【owner D】` 改触发:不再倒计时、不再轮圈)**:谁在自己回合里**改动了某个含 joker 的组**(增、删、拆、挪、取回 joker——任何改动),就在**结算时按概率掷一次引信**:`【owner】` **首次 20%,每多动一次 +15%,封顶 80%**(push-your-luck,越捅越烫;详见 §5)。
- **爆炸**:`【owner】` 引信中了 → 这个**组连同小丑整个炸散回牌库**,**触发改动的那个人摸 3 张**。
- **张力**:小丑组成了桌面的**危险区**——动它可能帮你拆出好料,也可能炸你一脸。种下小丑=给全场埋雷,你自己和任何想动这组的人都得掂量"敢不敢碰";想害人就引诱别人去动它。
- `【补】` 边角:两张小丑=两个独立危险组;**只放着不动它=不会自己炸**(没倒计时);"改动"的精确定义、p、是否累加、初次种下是否掷,都是旋钮。
- **可行性彩蛋**:这版**不需要任何倒计时/服务器定时器**——就是出牌结算时掷一次概率,完全落在回合制里(比倒计时还好做)。

## 5. 数值初版 v1(已锁 + 旋钮)

**已锁(`【owner】` 拍板):**
- **小丑引信**:种下不掷;之后**每动一次该 joker 组**,结算掷一次,**首次 20%,每多动一次 +15%,封顶 80%**;中了=**该组散回牌库 + 触发者摸 3**;joker 取回手再打 = 新雷、热度归零。
- **诈唬**:质疑成功 → 质疑者**消灭自己 1 张手牌**(回库,奖励)、诈唬人**摸 2** + 卡作废;质疑失败 → 质疑者**摸 2**。指定卡只目标质疑、非指定卡全场质疑。
- **全局**:一切"因惩罚而摸的牌"**只摸普通牌、不摸能力卡**。
- **发卡**:开局 2 张;每回合开始 **30%** 发 1 张;整副 **28 张**(白 13 / 蓝 9 / 金 6,配比见 §3)。
- **封族**锁 **2** 回合;**`触发②`阈值** = 贡献 **> 7** 张。

**首测初版(可调旋钮):**
- **转盘对象**:某玩家 50% / 公共牌桌 35% / 所有玩家 15%。
- **转盘·玩家事件**:摸 1–3 张 **45%**(1:40%/2:40%/3:20%)/ 丢回手里 1 张 **30%** / 整手重洗拿等量 **25%**。
- **转盘·牌桌事件**:加一套 50% / 收一套 50%。
- 仍可调:小丑引信的累加步长/封顶、初次种下是否掷、"改动 joker 组"精确定义;转盘各概率细调;`触发③④`(四色组 / 手牌≤3)是否开启;deck 大小 / 发卡率是否再调。

## 6. 明确不做 / 已澄清
- `【owner】` **转盘只做一个**;**不做秒级互动 / 微决策小游戏**(线上难做)。
- `【owner】` **不做隐藏身份 / 阵营(E)**——所有人目的一致。
- 不做实时手速、不做反应窗口(沿用回合制;护盾/转移/质疑都是"留身上或一次确认、结算那刻判",超时默认放过)。
- 普通牌**无弃牌堆、无销毁**;弃牌堆只给能力卡。
- 不设砸牌链上限、不设能力卡手上限、不公开能力卡张数、每回合不限出卡数(均 owner 拍板)。
